# TaskbarX
TaskbarX FOR WIN7

源自：http://www.tbtool.cc

自己增加的代码略拙劣，但基本上实现了想要的功能！

注意：
1. 可能只适用于WIN7
2. 需要管理员权限运行
3. 如果流量监控无效，请在窗口上点击右键，进入“功能设置”修改网络类型，直到有效！
