# TaskbarX
TaskbarX FOR WIN7

#### 感谢：
- 本程序基于图拉丁硬件检测源码：www.tbtool.cc [秃顶的小P孩/86738619]

#### 功能：
- CPU/GPU/内存占用
- CPU/GPU温度
- 上行/下行网络速度
- 实时测试IP/网站延迟[PING/TCPING]
- 今日天气/语音播报[每小时刷新]
- 百度实时热点[每小时刷新]

#### 注意：
- 最终运行效果只在WINx64位下测试
- 可能需要管理员权限运行
- 如果流量监控无效，请在窗口上点击右键，进入“功能设置”修改网络类型，直到有效！
