# TaskbarX

#### 功能：
- CPU/GPU/内存占用
- CPU/GPU温度
- 上行/下行网络速度
- 实时测试IP/网站延迟[PING/TCPING]
- 今日天气/语音播报[每小时刷新]
- 百度实时热点[每小时刷新]
- 可更换式LOGO
- 其他/隐藏功能请自行摸索/编辑配置

#### 注意：
- 可能需要管理员权限运行
- 最终运行效果只在WINx64位下测试
- 温度检测可能不支持较新的硬件，此类问题不需要反馈，无力解决
